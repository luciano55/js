Números: Number

  • Video 07
  • Number es un Object
  • Js tiene poca presición numérica. Para ello usar librerias especificas
  • Se pude crear mediante asignación directa: a = expresión;
  • Se pude crear mediante constructor: new Number(expresión);
  • Se pude crear mediante función: Number(expresión)
  • Si el argumento no puede ser convertido a un número, devuelve NaN.
  • Función isNaN para determinar si es número en caso contrario devuelve NaN
  • Important! el operador typeof
  • parseInt y parseFloat son funciones de alto nivel y están asociadas al objeto Function.
